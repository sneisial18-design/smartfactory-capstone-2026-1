### 3.2 프로젝트 요약

본 프로젝트는 단일 RGB-D 기준 영상(anchor image) 한 장을 이용하여 처음 보는 물체의 6D 자세를 추정하는 것을 목표로 하였습니다. 6D 객체 자세 추정은 물체의 3차원 회전과 위치를 추정하는 기술로, 로봇 그리핑, 증강현실, 스마트팩토리 자동화 등에 활용될 수 있습니다.

기존 베이스라인인 Any6D는 단일 이미지로부터 3D 메시를 생성한 뒤, 렌더링 결과와 실제 이미지를 비교하여 물체의 자세를 추정합니다. 그러나 단일 이미지 기반 3D 생성은 보이지 않는 뒷면을 추정해야 하는 어려움이 있어, 생성된 메시가 부정확해질 수 있습니다. 이 경우 부정확한 형상이 자세 추정 성능의 한계로 작용합니다.

이를 해결하기 위해 본 프로젝트에서는 생성 메시를 고정된 입력으로 사용하지 않고, 미분 가능한 3D Gaussian Splatting 표현으로 변환한 뒤 형상과 자세를 함께 정제하는 방법을 적용하였습니다.

---

### 3.3 연구 목표

* 단일 이미지 기반 3D 메시 생성의 부정확성 문제를 완화한다.
* 생성 메시를 고정된 입력이 아니라 함께 최적화하는 변수로 재정의한다.
* 형상(shape)과 자세(pose)의 상호 의존성을 반영하여 공동 정제를 수행한다.
* 3DGS 기반 미분 가능 렌더링을 이용해 이미지 오차로부터 형상 및 자세의 수정 방향을 계산한다.
* 기존 Any6D 대비 6D 객체 자세 추정 성능을 개선한다.

---

### 3.4 연구 방법

본 프로젝트의 전체 방법론은 3단계 파이프라인으로 구성됩니다.

#### Stage 1. Base Estimator

* 기준 영상(anchor image)으로부터 초기 3D 메시를 생성합니다.
* 생성된 메시를 query 이미지와 비교하여 초기 상대 자세를 추정합니다.
* 이 단계에서는 Any6D 기반의 초기 자세 추정 결과를 사용합니다.

#### Stage 2. Mesh-to-3DGS 변환

* 기존 메시 표현은 이산적인 표면 구조이므로 직접적인 gradient 기반 형상 갱신이 어렵습니다.
* 이를 해결하기 위해 메시를 3D Gaussian Splatting 표현으로 변환합니다.
* 표면 점을 샘플링하고, 각 점에 위치, 공분산, 불투명도, 색상 등의 Gaussian 속성을 부여합니다.

#### Stage 3. Joint Shape-Pose Refinement

* 미분 가능 렌더러를 이용하여 3DGS를 RGB, 깊이, 실루엣 형태로 렌더링합니다.
* 렌더링 결과와 실제 관측 이미지 사이의 오차를 계산합니다.
* Shape Step에서는 자세를 고정하고 형상을 갱신합니다.
* Pose Step에서는 형상을 고정하고 자세를 갱신합니다.
* 두 단계를 반복하여 최종 형상과 자세를 함께 정제합니다.

손실 함수는 다음 요소들로 구성됩니다.

* 광도 손실: L1 Loss
* 구조적 유사도 손실: MS-SSIM
* 실루엣 손실: Silhouette Loss
